## 🚀 How It Works

### 1. Campus Selection
- Click the location icon in header
- Modal displays 6 campus options
- Select your campus
- Campus name updates in header

### 2. Add to Cart
- Browse and add products
- Cart count updates
- Add multiple items

### 3. Checkout Process
- Click checkout button
- Order modal opens automatically
- Shows campus, items, time, total
- Click "Place Order" or "Cancel"

### 4. Order Tracking
- Live animation starts
- Shows 5 delivery stages
- Progress bar fills
- Pulsing delivery icon
- Auto-closes after completion
